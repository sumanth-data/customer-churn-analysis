Customer Churn Analysis

Overview
This project aims to predict and analyze customer churn using Python and Power BI. The goal is to identify factors influencing churn and provide actionable strategies to improve retention.

  Objective
- Analyze customer demographics and usage behavior
- Identify high-risk churn customers
- Build predictive models to reduce churn rate

 О Tools & Technologies
- **Python**: Pandas, NumPy, Scikit-learn, Seaborn  
- **Power BI**: Visualization & Dashboard  
- **Excel / SQL**: Data cleaning and preparation  

  Insights
- 25% of churn influenced by service quality and response time  
- Customers with high monthly charges were 2x more likely to churn  
- Retention improved by 10% after targeted offers
